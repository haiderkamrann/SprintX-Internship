This is tutorial of how we can select elements using different methods in DOM.

getElementById

In this method we choose an element by giving it a unique id.

getElementByClassName

In this method we choose an element by giving it a class. It returns a HTML Collection, which is also known as live collection

getElementsByTagName

We can choose elements using TagName, it will select all the elements matches. It returns a HTML Collection, which is also known as live collection

querySelector

In this method the querySelector chooses the element which occur first like if there is 2 h5 in my page and I select it using querySelector than it will select the first h5. It returns a NodeList containing all the elements that match the specified selector.

querySelectorAll

This method is used to select all the elements. It returns a NodeList which is not a live collection
